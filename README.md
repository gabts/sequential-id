# Sequential ID

A stupidly simple sequential ID generator.

## Installation

`npm i sequential-id`

## Example

```js
const sequentialId = require("sequential-id");
sequentialId(); // 00001
sequentialId(); // 00002
sequentialId(); // 00003
// ...
sequentialId(); // 9999999 and so on...
```
